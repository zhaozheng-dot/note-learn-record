# Note Learn Record

学习笔记过程记录库。记录每篇笔记的学习过程、实践验证和掌握状态。

## 与其他知识库的关系

```
note-learn-record/          ← 你在这里
├── scienc-project-repo/    ← 双链 → scienc-project-repo
├── science-tech-repo/      ← 双链 → science-tech-repo  
├── science-business-reop/  ← 双链 → science-business-reop
└── social-life-repo/       ← 双链 → social-life-repo
```

**双向链接机制：**
- 学习记录 → 知识库笔记：通过 `[[知识库/笔记名]]` 引用源笔记
- 知识库笔记 → 学习记录：源笔记底部追加 `> 学习记录：[[笔记名-learn-record]]`

## 命名规则

- 学习记录文件名：`笔记名-learn-record.md`
- 放在对应知识库的子目录下
